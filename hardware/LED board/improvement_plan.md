# LED基板改良計画

## 概要
本ドキュメントは、GPIO駆動PWM制御を主軸としたLED基板の改良計画を示します。電流管理と安全性に重点を置いています。

---

## 1. 基本構成

### 推奨回路構成

```
Raspberry Pi GPIO (PWM)
          │
        100Ω
          │
        Gate
        MOSFET (IRLML6344)
          │
          ├─ 10kΩ → GND (pull-down)
          │
        Source → GND
          │
        Drain
          │
        LED列
          │
        電流制限抵抗
          │
        12V
```

### 主要部品と役割

| 部品 | 役割 |
|------|------|
| 100Ω | GPIO保護 / スイッチング安定 |
| 10kΩ | 起動時OFF保証 |
| 抵抗 | LED電流制限（必須） |

## 3. LED電流設計

### LED仕様 (OSI5LAS1C1A)

| パラメータ | 値 |
|------------|----|
| IF (DC) | 100 mA |
| IF (パルス) | 1 A |
| 最大デューティ | ≤ 10% |

### 安全設計パラメータ

- パルス電流: 300–500 mA
- デューティ: 10%

---

## 4. 抵抗値推奨値

### LED構成: 6個直列

- **順電圧 (Vf)**: 1.6V × 6 = 9.6V
- **電源電圧**: 12V
- **余剰電圧**: 12V − 9.6V = 2.4V

### 電流計算

| 抵抗値 | 電流 |
|--------|------|
| 24Ω | 100 mA (現状) |
| 6Ω | ≈400 mA |
| 4Ω | ≈600 mA |

### 推奨値

- **5–6Ω**: パルス電流と安全性の最適バランス

---

## 5. 電力消費解析

### 現状構成

- パルス: 400 mA (デューティ 10%)
- 平均電流: 40 mA
- 4列: 160 mA (平均)
- **電力**: 12V × 0.16A ≈ 2W

### 比較

- **現状**: ≈4W
- **改良後**: ≈2W
- **削減**: 50%

---

## 6. 重要な安全注意事項

### 抵抗の必須性

LEDは指数特性を示す。抵抗なしでは:
- 少し電圧上昇 → 電流暴走 → LED破壊

**結論**: 電流制限抵抗は、パルス駆動でも絶対に必要。

## 7. Python PWM例

import RPi.GPIO as GPIO

GPIO.setup(PIN, GPIO.OUT)

pwm = GPIO.PWM(PIN, 500)
pwm.start(10)
